# BackstageOS - User Features Overview

## Core Platform

### Show Management
- Show-centric organization - Everything organized by production with complete separation between shows
- Project lifecycle tracking with 7 key production milestones (Prep Start, First Rehearsal, Designer Run, First Tech, First Preview, Opening Night, Closing)
- Team collaboration - Invite team members with role-based access
- Secure sharing - Generate shareable links for external collaborators

### User Profiles
- Profile types - Freelance vs Full-time theater professionals with tailored workflows
- Personal settings and preferences
- Profile management with contact information

## Production Tools

### Report System
- Four report types - Rehearsal, tech, performance, and meeting reports
- Custom template builder with drag-and-drop field arrangement
- Rich text editor with Google Docs-style interface and formatting
- Phase-specific templates for different production stages
- Auto-save functionality with visual status indicators
- Professional formatting with page numbering and clean layouts

### Script Editor
- Collaborative editing with real-time updates like Google Docs
- Theater script formatting with proper stage play conventions
- File import - Upload .txt, .rtf, .html, .docx, .doc, and .pdf files
- Version control - Complete history with ability to publish and revert changes
- Smart page management with automatic content flow
- Comments system with inline commenting and threaded discussions
- Auto-formatting for play elements (titles, character names, stage directions)
- Cue management for lighting, sound, video, and automation cues

### Props & Costume Tracking
- Props tracker - Organize by scene/character with status tracking and sourcing notes
- Costume tracker - Track quick-change timing, repairs, and character assignments
- Image uploads for visual reference
- Status management (needed, pulled, rehearsal, performance, returned)
- Scene and character organization

### Contact Management
- Complete contact system with team and company organization
- Contact sheets with professional formatting and role filtering
- Availability tracking with visual drag-and-drop interface
- Team member management for each production

## Scheduling & Calendar

### Schedule Management
- Weekly schedule view with drag-and-drop event creation
- Team availability with visual availability tracking
- Location management for venues and rehearsal spaces
- Flexible time increments (15, 30, 60 minutes)
- Event categories with customizable types
- Conflict detection to prevent scheduling overlaps
- Daily call sheet generation with professional formatting
- Public calendar sharing for cast and crew access

### Notifications
- Schedule change alerts when events are modified
- Email notifications for important updates
- Production milestone reminders

## Notes & Task Management

### Notes System
- Rich text editing with comprehensive formatting options
- Folder organization with drag-and-drop reordering
- Search and tagging across all notes
- Real-time collaboration with sharing and comments
- Auto-save with visual save status
- Dual views - Global notes and show-specific notes
- Pin important notes and archive completed items
- File attachments for documents and media

### Task Management
- List and board views for different workflow preferences
- Custom task properties and metadata
- Team assignment and tracking
- Task comments and collaboration
- File attachments for task-related documents
- Progress tracking through production phases

## Performance & Rehearsal Tracking
- Performance metrics specific to each show
- Rehearsal progress monitoring
- Production milestone tracking through all phases
